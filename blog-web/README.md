# app

ウェブアプリ

## 開発環境

必須:

- Node.js (>= 10)

推奨:

- Git
    - Windows の場合、シェルスクリプトの実行に Git Bash を使うのを推奨
- VSCode
    - 拡張機能:
        - ESLint

### ビルドタスク

開発用:

- `./dev`: 開発モードでのサーバーの起動やテストの実行など
- `./fix`: セミコロンや末尾カンマの抜けなどを自動修正する
- `./check`: コンパイルエラーを表示する

本番用:

- `./prod`: 本番モードでのサーバーの起動
- `./test`: 本番モードでのテスト実行
